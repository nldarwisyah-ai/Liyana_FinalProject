# Liyana_FinalProject
Hand Tetris is a game where you play Tetris using your hands instead of a keyboard. The webcam tracks your hand in real time. When you open your palm and move left or right, the block moves with you. When you clench your fist, the block drops. If you turn your wrist while your palm is open, the block rotates. There are also coloured indicators on screen to show what action is being detected.

The game layout is simple, with a tall Tetris board, a camera preview, and an instruction panel. The goal is the same as normal Tetris, which is to clear lines and not let the blocks reach the top.

During this project, I struggled a lot with my laptop. It kept lagging and sometimes crashing when I used VS Code. It was very stressful because the game would not run properly and it slowed down my progress.

After fixing and simplifying the code, the game became much smoother and easier to use. Now it runs well with less lag, and I am really happy and satisfied with how it turned out.
